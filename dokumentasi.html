Buku Panduan BejajaCafePOS

Sistem Kasir Kafe Premium Offline-First PWA

1. Panduan Penggunaan Aplikasi

A. Login & Hak Akses

  • Saat pertama kali dibuka, sistem dilindungi oleh PIN 6 angka.
  • Admin Default: PIN 123456 (Memiliki akses penuh ke Laporan Keuangan, Pengaturan, dan Manajemen User).
  • Kasir Default: PIN 111111 (Hanya bisa mengakses Kasir, Menu, dan Riwayat Transaksi).

B. Melakukan Transaksi (Kasir / POS)

  1. Buka menu Kasir.
  2. Pilih kategori atau cari nama menu di kolom pencarian, lalu klik gambar/nama menu yang dipesan.
  3. Di area keranjang (sisi kanan pada PC, atau klik tombol tas belanja pada HP), sesuaikan jumlah pesanan dengan tombol + atau -.
  4. Klik tombol Lanjut Pembayaran.
  5. Pilih Metode Pembayaran (Tunai, QRIS, Debit, atau Kredit). Untuk tunai, Anda bisa menggunakan tombol cepat nominal uang (50k, 100k) atau ketik manual.
  6. Klik Konfirmasi Pembayaran. Kembalian akan otomatis terhitung.

Penting: Transaksi yang belum dibayar di keranjang tidak akan hilang meski Anda tidak sengaja me-refresh halaman (Fitur Anti-Data Loss).

C. Mengelola Menu & HPP (Harga Pokok Penjualan)

  • Masuk ke menu Menu Kafe (Hanya Admin).
  • Klik Tambah Menu.
  • Isi Nama Menu, Kategori, Harga Modal (HPP), dan Harga Jual. Catatan: Mengisi HPP sangat penting agar sistem bisa menghitung Laba Bersih secara otomatis di Laporan Keuangan.
  • Upload gambar langsung dari perangkat atau gunakan URL gambar dari internet. Klik Simpan.

D. Laporan Keuangan & Export Excel (Khusus Admin)

  • Buka menu Riwayat, lalu klik tombol biru Lihat Laporan Keuangan di pojok kanan atas.
  • Sistem akan menampilkan ringkasan cerdas: Total Omzet, Total Modal, Laba Bersih, dan grafik interaktif performa penjualan 7 Hari Terakhir.
  • Untuk pembukuan yang lebih detail, gunakan fitur Export Excel. Anda bisa langsung mengunduh data hari ini, minggu ini, bulan ini, atau memilih rentang tanggal spesifik (Custom). File Excel akan otomatis berisi rincian HPP dan Laba Bersih per transaksi.

2. Panduan Pengaturan & Kustomisasi

A. Auto-Save Pengaturan

Fitur terbaru di BejajaCafePOS: Anda tidak perlu lagi mencari tombol "Simpan" di menu Pengaturan. Setiap kali Anda mengubah warna tema, mengganti nama aplikasi, mengetik alamat struk, atau mengunggah logo, sistem akan menyimpannya secara otomatis dan memunculkan notifikasi hijau di pojok kanan atas.

B. Mengatur Pembayaran QRIS

Di Pengaturan, Anda dapat memunculkan QR Code toko Anda untuk mempermudah pelanggan. Sangat disarankan untuk menggunakan mode URL Link saat mengunggah gambar QRIS. Dengan menggunakan link, gambar tidak akan hilang walau Anda melakukan backup/restore atau reset aplikasi.

C. Sinkronisasi Google Sheets

Untuk keamanan ekstra, koneksikan aplikasi ke Google Sheets. Masukkan URL Web App Google Script di pengaturan. Setiap transaksi sukses akan langsung di-backup ke cloud. Jika kasir sedang offline (tidak ada internet), data akan disimpan sementara di perangkat dan Anda bisa menekan tombol Sinkron Sekarang saat internet kembali stabil.

3. Panduan Hosting & Instalasi PWA

Aplikasi ini dibuat murni menggunakan HTML, CSS (Tailwind), dan Javascript (IndexedDB), sehingga tidak membutuhkan server backend/database yang mahal. Anda dapat menghostingnya secara GRATIS.

Menggunakan GitHub Pages (Gratis & Disarankan)

  1. Buat akun di GitHub.
  2. Buat repository baru, misalnya dengan nama BejajaCafePOS-app.
  3. Upload file index.html, manifest.json, sw.js, dan buat folder assets/ lalu upload gambar logo Anda (harus bernama icon-192x192.png & icon-512x512.png).
  4. Pergi ke tab Settings pada repository tersebut -> Pilih Pages (di sidebar kiri).
  5. Pada bagian Source, pilih Deploy from a branch, pilih branch main, dan klik Save.
  6. Tunggu beberapa menit, GitHub akan memberikan link website HTTPS. Buka link tersebut di Tablet/HP Kasir, lalu pilih opsi "Add to Home Screen" (Instal Aplikasi) agar BejajaCafePOS berjalan penuh tanpa border browser layaknya aplikasi Android/iOS asli.
